API grafiki
Zmienia biliotekę grafiki 3D, która jest wykorzystywane przez grę.\nJako że jest to eksperymentalna opcja, zalecane jest pozostawienie jej na "Auto". Zmiana może pomóc w przypadku pewnych problemów z renderowaniem lub wydajnością sprzętu o gorszym wsparciu.
Zmiany zajdą po ponownym uruchomieniu gry.
Niedostępne, ponieważ ustawienie API grafiki zostało wymuszone w opcjach uruchamiania gry.
